# My Website

This is my personal website, built with Next.js 14 and Tailwind CSS. It features a simple, modern design and includes the following sections:

- **Home**: A brief introduction to who I am and what I do.
- **About**: A more detailed overview of my background, skills, and experience.
- **Projects**: A showcase of some of my favorite projects, with links to the code and live demos.
- **Experience**: A list of my work experience, including job titles, company names, and dates.
- **Skills**: A list of my skills, including programming languages, frameworks, and tools.
- **Publications**: A list of my publications, including research papers and articles.
- **Contact**: A simple contact form that allows visitors to get in touch with me.
